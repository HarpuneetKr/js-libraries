# Discover Toronto

A responsive landing page showcasing Toronto's top attractions, neighbourhoods, and experiences.

## Installation

```bash
npm install
npm run start
```

This starts a local dev server at `http://localhost:3000`.

## Libraries Used

| Library                                                                | Version | Purpose                             |
| ---------------------------------------------------------------------- | ------- | ----------------------------------- |
| [GSAP](https://gsap.com/)                                              | 3.15.0  | Animations & ScrollTrigger effects  |
| [Lenis](https://lenis.darkroom.engineering/)                           | 1.3.25  | Smooth scrolling                    |
| [Swiper](https://swiperjs.com/)                                        | 14.0.1  | Image carousel/slider               |
| [Lucide](https://lucide.dev/)                                          | 1.23.0  | Icons                               |
| [Playfair Display](https://fonts.google.com/specimen/Playfair+Display) | —       | Serif heading font (Google Fonts)   |
| [DM Sans](https://fonts.google.com/specimen/DM+Sans)                   | —       | Sans-serif body font (Google Fonts) |
